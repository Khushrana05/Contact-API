# 📇 Contact API

A RESTful Contact Management API built using Node.js, Express.js, MongoDB, and JWT Authentication.

## 🚀 Features

- User Signup
- User Login (JWT)
- Save Contact
- Get All Contacts
- Get Contact By ID
- Update Contact
- Protected Routes

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
- dotenv
